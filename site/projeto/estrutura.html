Estrutura e Função

Bancada Festo OSR241T controlada por ESP32

Módulos da Bancada

  • Módulo de Alimentação: Interface com ESP32 e fonte 24V/5V
  • Módulo de Atuadores: Cilindros pneumáticos dupla ação
  • Módulo de Sensores: Fim de curso, indutivos, ópticos
  • Módulo de Controle: ESP32 com shield de relés
  • Módulo de IHM: Interface web via Wi-Fi
ESP32: Central de processamento

Funções Principais

  • Separação de peças: Controle de cilindros por sensores
  • Transporte: Atuação sequencial pneumática
  • Identificação: Leitura de sensores (peças metálicas/não metálicas)
  • Classificação: Lógica programada no ESP32
  • Monitoramento: Dashboard web em tempo real

Especificações Técnicas

Pressão de trabalho: 600 kPa (6 bar)

Alimentação: 24V / 5V (ESP32)

Controlador: ESP32-WROOM-32

Conectividade: Wi-Fi 2.4GHz, Bluetooth

Memória: 4MB Flash, 520KB RAM

Clock: 240 MHz

Estrutura Física

Dimensões: 800mm x 600mm

Peso: Aprox. 25kg

Materiais: Alumínio, aço inox
